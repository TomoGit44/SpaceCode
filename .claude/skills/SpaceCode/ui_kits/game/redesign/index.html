設計上の前提

  • 配色 既存 5 色を維持。新色追加なし。彩度のみ調整。
  • 演出 7/10 — グロー (filter: drop-shadow) + パーティクル + 衝撃波 + 短い余韻。爆発は 3 層リング。
  • 実装 HTML/CSS/SVG/JS のみ。Phaser 本体への移植は別途検討 (グロー → BlurFX / radialGradient → makeFilledTextureShape)。
  • スコープ 各領域 1 案決め打ち。レビュー後に絞り込み・微調整。

次のステップ

  • 各画面のレビュー → 採用部分の確定

  • 必要なら細部 (色味、タイミング、密度) の調整版を 1〜2 ターン
